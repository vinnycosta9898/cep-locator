# 💻 Project:
Cep-Locator é uma aplicação frontend que consome dados de uma API e exibe o endereço de CEP na tela.


# 🚀 Technologies:
### ✔️ Axios
### ✔️ ReactJS
### ✔️ Vite

# How to run

```
# Clone this repository
$ git clone https://github.com/vinnycosta9898/cep-locator

# Go to the directory
$ cd cep-locator

# Install Dependencies
$ npm install

# Run Web Server
$ npm run dev
